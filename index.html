Frequently Asked Questions

What is Matrix?
Matrix is an open standard for secure, decentralized, real-time communication. It allows you to communicate with anyone, regardless of which app they use.
Is QuikxChat free?
Yes! QuikxChat is completely free and open source under the AGPL-3.0 license.
How secure is QuikxChat?
QuikxChat uses end-to-end encryption (E2EE) by default for all messages, ensuring only you and your recipient can read them.
Can I use my existing Matrix account?
Absolutely! QuikxChat works with any Matrix homeserver, so you can use your existing account or create a new one.
